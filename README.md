# Regression Project

## Overview

This project demonstrates different Regression algorithms using Python and Scikit-learn. It includes:

- Simple Linear Regression
- Multiple Linear Regression
- Polynomial Regression

Each model performs data preprocessing, training, prediction, evaluation, visualization, and user input prediction.

## Features

### Common Features

- Load dataset from CSV file
- Handle missing values
- Encode categorical variables (if present)
- Select feature(s) and target column
- Split dataset into training and testing sets
- Train regression model
- Predict target values
- Calculate R² Score
- Accept user input for prediction

### Simple Linear Regression

- Train a Simple Linear Regression model
- Plot regression line with data points
- Predict values using one independent variable

### Multiple Linear Regression

- Train a Multiple Linear Regression model
- Work with multiple independent variables
- Predict target values using multiple features

### Polynomial Regression

- Transform features into polynomial features
- Train Polynomial Regression model
- Compare Linear vs Polynomial regression curve
- Visualize polynomial fitting
